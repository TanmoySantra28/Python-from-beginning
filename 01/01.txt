Print statement, comments, Escape sequence

print : print is used to print or display anything.
"\n" : escape sequence character, used to print the line into next line.
# : comment a single-line of code.
""" """ or ''' ''' : multi-line comment.
"\...\" : double quote character, to add double quote in a string.
sep="" : separator, separates objects within print statement, default is ''
end="" : specify what to print at the end, default is "\n"
